# 定格今日 · 复盘闪卡

**日拱一卒** — 每日复盘 + 领域复盘 + 自我报告，数据保存在浏览器 LocalStorage。

## 功能

- **领域管理**：工作、副业、生活、学习等，可自定义增删
- **今日复盘**：1-5 星评分 + 今日状态标签（一天一条，保存覆盖）
- **领域复盘**：按领域记录，支持五种类型（每日踩坑 / 关键行为 / 认知修正 / 灵感宝箱 / 情绪垃圾桶），标题 + 正文（可多条）
- **历史记录**：按复盘类型 / 按日 / 按领域三种回顾方式（同级切换），支持标为有价值、编辑、删除
- **自我报告**：周度（周一至周日）/ 月度 / 年度报告，统计今日复盘天数、各领域复盘次数与类型分布，展示本周期标为有价值的复盘
- **数据备份**：在「领域设置」页导出/导入 JSON 备份

## 本地运行

ES Module 需要通过本地服务器打开（不要直接双击 `index.html`）。

```bash
cd /path/to/my-cursor-app
npx serve .
```
